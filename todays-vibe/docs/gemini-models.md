## 🆓 무료 티어 (Free Tier) 한도

> Google AI Studio 기준. 프로젝트 설정 및 청구 계정 연결 여부에 따라 다를 수 있음.

| 모델 | RPM (분당 요청) | RPD (일일 요청) | TPM (분당 토큰) | 참고 |
|------|--------------|--------------|--------------|------|
| `gemini-1.5-flash` | 15 | 1,500 | 1,000,000 | v1beta deprecated → 사용 불가 |
| `gemini-2.0-flash-lite` | 30 | 1,500 | 1,000,000 | 이 프로젝트에서는 쿼터 0으로 나옴 |
| `gemini-2.0-flash` | 15 | 1,500 | 1,000,000 | 이 프로젝트에서는 쿼터 0으로 나옴 |
| `gemini-2.5-flash` | 10 | **20** | 250,000 | Preview 단계라 RPD가 매우 적음 |

> 하루 배치 1회(6 calls) + 사용자 꿈해몽 ~14회면 당일 한도 도달.

## 💳 유료 티어 (Pay-as-you-go) 가격

> 청구 계정 연결 시 적용. 단위: **1M 토큰당 USD**

| 모델 | 입력 토큰 | 출력 토큰 | 컨텍스트 윈도우 |
|------|----------|----------|--------------|
| `gemini-2.0-flash-lite` | $0.075 | $0.30 | 1M |
| `gemini-2.0-flash` | $0.10 | $0.40 | 1M |
| `gemini-2.5-flash` | $0.15 | $0.60 | 1M |
| `gemini-2.5-pro` | $1.25 (≤200K) / $2.50 (>200K) | $10.00 | 1M |

### 실제 비용 추산 (이 프로젝트 기준)

**배치 1회 (주간/월간/연간 중 하나):**
- 프롬프트 1개 ≈ 입력 2,000 토큰 + 출력 8,000 토큰
- 2 calls (별자리 + 띠별) × 10,000 토큰 = 약 20,000 토큰
- `gemini-2.5-flash` 기준: 약 **$0.006 (₩8원)** per 배치

**사용자 꿈해몽 1회:**
- 입력 500 + 출력 2,000 토큰 ≈ 2,500 토큰
- `gemini-2.5-flash` 기준: 약 **$0.00145 (₩2원)** per 해몽

**월 규모 추산 (서비스 오픈 후):**

| 사용자 규모 | 꿈해몽 일 100회 | 배치 월 12회 (주간×4+월간×1+연간×1×2) | 월 예상 비용 |
|------------|--------------|--------------------------------------|-------------|
| 소규모 (~100명) | 100 × $0.00145 × 30 = $4.35 | 12 × $0.006 = $0.07 | **약 $4.5 (₩6,300원)** |
| 중규모 (~1,000명) | 1,000 × $0.00145 × 30 = $43.5 | $0.07 | **약 $44 (₩62,000원)** |
| 대규모 (~10,000명) | 10,000 × $0.00145 × 30 = $435 | $0.07 | **약 $435 (₩610,000원)** |

> 📌 배치(스케줄 운세)는 사용자 수와 무관하게 고정 비용. 변동 비용의 핵심은 **AI 해몽/사주 등 개인화 기능**.

## ⚖️ 모델별 장단점

### `gemini-2.0-flash-lite`
| 장점 | 단점 |
|------|------|
| 가장 저렴 ($0.075/1M) | 품질이 낮음 |
| RPM 30으로 빠른 병렬 처리 | 긴 문맥 이해 약함 |
| 응답 속도 빠름 | 이 프로젝트에서 쿼터 0 |

> 💡 대량 배치, 품질보다 속도/비용이 중요한 경우 적합

---

### `gemini-2.0-flash`
| 장점 | 단점 |
|------|------|
| 가격 대비 성능 균형 | 2.5-flash 대비 품질 낮음 |
| 무료 RPD 1,500 (조건 충족 시) | 이 프로젝트에서 쿼터 0 |
| 안정적 (Preview 아님) | — |

> 💡 유료 전환 후 배치 생성에 적합한 대안

---

### `gemini-2.5-flash` ✅ 현재 사용 중
| 장점 | 단점 |
|------|------|
| 현재 최고 실용 모델 | 무료 RPD 20 (매우 적음) |
| 꿈해몽, 운세 품질 우수 | Preview → 가격 정책 변동 가능 |
| 긴 컨텍스트 처리 좋음 | RPM 10으로 병렬 제한 |
| 스트리밍 응답 지원 | — |

> 💡 사용자 대면 AI 기능 (꿈해몽, 사주 등)에 최적. 유료 전환 시 배치도 이걸로 통일 가능.

---

### `gemini-2.5-pro`
| 장점 | 단점 |
|------|------|
| 최고 품질 응답 | 매우 비쌈 ($10/1M 출력) |
| 복잡한 추론 우수 | 무료 RPD 25 (테스트 수준) |
| 1M 컨텍스트 윈도우 | 응답 느림 |

> 💡 정밀한 사주/궁합 해석 프리미엄 기능에 고려 가능. 현재는 사용 불필요.

## 🛣️ 이 프로젝트 권장 전략

### 지금 (개발/테스트 단계)
```
배치 생성  → gemini-2.5-flash (하루 1회만, RPD 아껴서 사용)
꿈해몽     → gemini-2.5-flash (테스트는 최소한으로)
```

### 서비스 오픈 전 (유료 전환 후)
```
배치 생성  → gemini-2.0-flash-lite (저렴, 배치는 품질보다 속도)
꿈해몽/사주 → gemini-2.5-flash     (사용자 대면, 품질 중요)
```

### 프리미엄 기능 추가 시 (미래)
```
일반 해몽   → gemini-2.5-flash
정밀 사주   → gemini-2.5-pro  (유료 플랜 사용자만)
```

## 🔧 유료 전환 방법

1. [Google Cloud Console](https://console.cloud.google.com) → 결제 계정 연결
2. Gemini API 사용량 알림 설정 (예산 초과 방지)
3. `.env.local` `GEMINI_API_KEY` 는 동일하게 유지

> 💡 월 $10 미만이면 Gemini 무료 크레딧으로 처리되는 경우도 있음 (신규 GCP 계정 $300 크레딧 제공).
