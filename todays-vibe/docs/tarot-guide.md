# 타로 카드 가이드

타로 카드의 구성, 각 카드 의미, 점술 방법을 정리한 개발 참고 문서.

---

## 1. 타로 덱 구성 (총 78장)

```
78장 = 메이저 아르카나 22장 + 마이너 아르카나 56장
```

---

## 2. 메이저 아르카나 (Major Arcana) — 22장

삶의 큰 흐름·운명적 사건·영적 여정을 나타낸다.  
번호 0~XXI로 구성되며, 0(광대)은 여정의 시작이자 끝을 상징한다.

| 번호 | 영문 | 한국어 | 키워드 | 정방향 의미 | 역방향 의미 |
|------|------|--------|--------|------------|------------|
| 0 | The Fool | 광대 | 새 시작, 순수, 모험 | 새로운 출발, 자유, 잠재력 | 무모함, 경솔, 준비 부족 |
| I | The Magician | 마법사 | 의지, 능력, 실현 | 의지력 발휘, 자원 활용, 창조 | 조작, 능력 낭비, 속임수 |
| II | The High Priestess | 여사제 | 직관, 비밀, 내면 | 직관을 따르라, 숨겨진 진실 | 표면적 정보에 의존, 비밀 드러남 |
| III | The Empress | 여황제 | 풍요, 모성, 창조 | 풍요로움, 출산, 자연의 힘 | 의존, 과잉 보호, 창작 막힘 |
| IV | The Emperor | 황제 | 권위, 구조, 안정 | 리더십, 안정된 기반, 질서 | 독재, 경직, 통제 과잉 |
| V | The Hierophant | 교황 | 전통, 제도, 신념 | 관습 따르기, 멘토의 조언 | 반항, 비전통, 규칙 거부 |

| XXI | The World | 세계 | 완성, 통합, 성취 | 목표 달성, 완전한 성취 | 미완성, 지연, 정체 |

## 3. 마이너 아르카나 (Minor Arcana) — 56장

일상의 사건·감정·상황을 나타낸다.  
4개 수트 × 14장 (Ace~10 숫자 카드 10장 + 코트 카드 4장).

| 수트 | 원소 | 상징 영역 | 현대 트럼프 대응 |

| 펜타클 (Pentacles/Coins) | 흙 🌍 | 물질·돈·건강·현실 | 다이아몬드 ♦ |

## 4. 정방향 / 역방향

카드를 뽑을 때 위아래 방향이 있다.

- **정방향 (Upright)**: 카드 본래 에너지가 온전히 발현
- **역방향 (Reversed)**: 해석 방식은 리더마다 다름
  - 에너지 약화 / 지연
  - 내면화 (외부→내부로 작용)
  - 카드 의미의 부정적 측면 강조
  - 차단 또는 과잉

역방향을 사용하지 않는 리더도 있다 (덱 설정에 따라 선택 가능).

## 5. 타로 점술 방법 (스프레드)

### 5-1. 원카드 (One Card)

```
[ 1 ]
```

- 카드 1장만 뽑는다
- "오늘 하루 에너지는?", "이 질문에 대한 핵심 메시지는?" 같은 단순 질문에 적합
- 초보자에게 추천

---

### 5-2. 쓰리카드 (Three Card Spread)

```
[ 1 ] [ 2 ] [ 3 ]
```

가장 많이 쓰이는 스프레드. 3장의 배치 의미는 질문 유형에 따라 달라진다.

| 배치 목적 | 1번 | 2번 | 3번 |
|----------|-----|-----|-----|
| 시간 흐름 | 과거 | 현재 | 미래 |
| 상황 분석 | 상황 | 행동 | 결과 |
| 관계 분석 | 나 | 상대방 | 관계 |
| 문제 해결 | 문제 | 조언 | 결과 |
| 선택 기로 | 선택 A | 선택 B | 조언 |

---

### 5-3. 켈틱 크로스 (Celtic Cross) — 10장

가장 전통적이고 복합적인 스프레드.

```
            [ 5 ]
[ 3 ] [ 1/2 ] [ 4 ]    [ 10 ]
            [ 6 ]       [ 9 ]
                        [ 8 ]
            [ 7 ]       [ 7 ]
```

실제 배치:
```
         [ 5 ]
[ 3 ] [1위에2] [ 4 ]    [ 10 ]
         [ 6 ]           [ 9 ]
         [ 7 ]           [ 8 ]
                         [ 7 ]
```

| 위치 | 의미 |
|------|------|
| 1 | 현재 상황 (핵심) |
| 2 | 교차 요소 (장애 or 도움) — 1 위에 가로로 올림 |
| 3 | 과거 영향 |
| 4 | 가까운 미래 |
| 5 | 의식적 목표 / 이상 |
| 6 | 잠재의식 / 두려움 |
| 7 | 질문자의 태도 |
| 8 | 외부 환경 / 주변 영향 |
| 9 | 희망 or 두려움 |
| 10 | 최종 결과 |

---

### 5-4. 말굽 스프레드 (Horseshoe) — 7장

```
[ 7 ]
[ 6 ]   [ 1 ]
[ 5 ]   [ 2 ]
[ 4 ] [ 3 ]
```

| 위치 | 의미 |
|------|------|
| 1 | 과거 |
| 2 | 현재 |
| 3 | 숨겨진 영향 |
| 4 | 장애물 |
| 5 | 주변 태도 |
| 6 | 취해야 할 행동 |
| 7 | 최종 결과 |

---

### 5-5. 관계 스프레드 (Relationship Spread) — 5장

```
[ 1 ]  [ 3 ]  [ 2 ]
       [ 4 ]
       [ 5 ]
```

| 위치 | 의미 |
|------|------|
| 1 | 나의 감정 / 상태 |
| 2 | 상대방의 감정 / 상태 |
| 3 | 관계의 현재 상태 |
| 4 | 관계의 장애물 |
| 5 | 관계의 방향 / 조언 |

## 6. 점술 진행 순서

```
1. 질문 설정
   → 구체적일수록 좋다. "미래는?" (X) / "올해 직장 운은?" (O)

2. 덱 섞기 (셔플)
   → 카드를 충분히 섞으며 질문에 집중

3. 스프레드 선택
   → 질문 유형에 맞는 스프레드 선택

4. 카드 뽑기
   → 스프레드 자리 수만큼 순서대로 뽑기

5. 카드 해석
   → 각 자리의 의미 × 카드 의미 × 정역방향 조합
   → 카드 간 관계(스토리)도 함께 읽기

6. 종합 해석
   → 전체 카드를 이어 하나의 이야기로 연결
```

## 7. AI 타로 해석 시 고려사항

### 입력 데이터
- 질문 (선택 사항 — 없으면 전반적 운세)
- 스프레드 종류
- 뽑힌 카드 목록 + 각 카드의 정/역방향
- 카드 위치(자리) 정보

### AI 프롬프트 구성 예시
```
스프레드: 쓰리카드 (과거-현재-미래)
질문: 올해 연애운

1번 (과거): The High Priestess (정방향)
2번 (현재): 3 of Swords (역방향)  
3번 (미래): The Star (정방향)

위 카드들을 타로 해석 전문가의 시각으로 각 카드의 의미를 자리와 연결하여
자연스러운 스토리로 해석해주세요. 한국어로 따뜻하고 통찰력 있게 작성해주세요.
```

### 해석 시 주의사항
- 의학·법률·금융 조언 금지
- 확정적 미래 예언 금지 ("~할 것이다" → "~의 가능성이 있다")
- 부정적 카드도 성장의 관점에서 해석
- 질문자가 행동 주체임을 강조

## 8. 덱 종류

| 덱 | 특징 |
|----|------|
| 라이더-웨이트 (Rider-Waite) | 가장 표준적, 모든 카드에 그림 | 
| 마르세유 (Tarot de Marseille) | 고전적, 마이너 아르카나는 문양만 |
| 토트 타로 (Thoth Tarot) | 카발라·점성술 결합, 고급 |
| 오쇼 타로 (Osho Zen) | 명상·선불교 철학 기반 |

> 이 프로젝트는 **라이더-웨이트** 기준으로 개발한다.

## 9. 카드 ID 규칙 (개발용)

`src/data/tarot-cards.json` 기준. 하이픈(-) 구분자 사용.

```
메이저: major-00 ~ major-21
마이너: {suit}-{number}
  - 수트: wands / cups / swords / pentacles
  - 숫자: ace, 02~10, page, knight, queen, king

예시:
  major-00       → The Fool (광대)
  major-01       → The Magician (마법사)
  cups-ace       → Ace of Cups (컵 에이스)
  swords-07      → 7 of Swords (소드 7)
  wands-king     → King of Wands (완드 킹)
```
