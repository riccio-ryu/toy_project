## 1. 주역 개요

**주역(周易)**은 서주(西周) 시대에 완성된 동양 최고(最古)의 점서(占書)이자 철학서다.  
원제는 **역(易)**, 경전화 이후 **역경(易經)**으로도 불린다.

| 구분 | 내용 |
|------|------|
| 저자 | 복희(괘 창제) → 문왕(괘사) → 주공(효사) → 공자(十翼 주석) |
| 구성 | 64괘 × 괘사 + 384효 × 효사 + 십익(十翼) 주석 10편 |
| 핵심 원리 | 음양(陰陽)의 변화 — 만물은 끊임없이 순환·변화한다 |

### 역(易)의 세 가지 원리

| 원리 | 설명 |
|------|------|
| **불역(不易)** | 변화 속에도 변하지 않는 불변의 법칙이 있다 |
| **간이(簡易)** | 만물은 음·양의 두 원리로 설명할 수 있다 |
| **변역(變易)** | 만물은 끊임없이 변화하며, 그 과정은 순환적이다 |

팔괘는 세 개의 효(爻)를 쌓아 만든 8가지 기본 괘다.  
아래→위 순서로 읽으며, 양효(─)=1, 음효(– –)=0.

| 기호 | 괘명 | 한자 | 이진 | 자연상 | 특성 |

| ☷ | 곤 | 坤 | 000 | 땅(地) | 수용, 순응, 모(母) |
| ☳ | 진 | 震 | 100 | 우레(雷) | 움직임, 시작, 장남 |

## 4. 효(爻)와 괘(卦) 구성

### 대성괘(大成卦) 구조
```
─── ← 6효 (상효, 上爻) ← 상괘(上卦, 외괘) 최상단
─── ← 5효
─── ← 4효
─── ← 3효 ← 하괘(下卦, 내괘) 최상단  
─── ← 2효
─── ← 1효 (초효, 初爻) ← 최하단
```

- **하괘(내괘)**: 1·2·3효 → 하단 삼효
- **상괘(외괘)**: 4·5·6효 → 상단 삼효
- 괘명 표기: `외괘(위) + 내괘(아래) + 괘명`  
  예: 地天泰 = 坤(위) + 乾(아래) = 태괘

### 효의 명칭
| 효 번호 | 위치 | 양효 | 음효 |
|---------|------|------|------|
| 초효(初爻) | 최하 | 초구(初九) | 초육(初六) |
| 이효(二爻) | 2번째 | 구이(九二) | 육이(六二) |
| 삼효(三爻) | 3번째 | 구삼(九三) | 육삼(六三) |
| 사효(四爻) | 4번째 | 구사(九四) | 육사(六四) |
| 오효(五爻) | 5번째 | 구오(九五) | 육오(六五) |
| 상효(上爻) | 최상 | 상구(上九) | 상육(上六) |

> 양효는 九(9), 음효는 六(6)으로 표기한다 (동전 점에서 유래).

## 7. 점치는 방법 (동전법)

### 동전 3개 던지기
동전 하나: 앞(陽)=1, 뒤(陰)=0

| 합계 | 효 종류 | 기호 | 의미 |
|------|---------|------|------|
| 3 (앞앞앞) | 노양(老陽) | ─○─ | 극양→음으로 변함 (변효) |
| 2 (앞앞뒤) | 소양(少陽) | ─── | 양효 (불변) |
| 1 (앞뒤뒤) | 소음(少陰) | ── ── | 음효 (불변) |
| 0 (뒤뒤뒤) | 노음(老陰) | ─×─ | 극음→양으로 변함 (변효) |

### 6효 뽑기 순서
1회 던지기 = 1효. **아래(1효)에서 위(6효)** 방향으로 6번 던진다.

```
6번째 던지기 → 6효 (상효)
5번째 던지기 → 5효
4번째 던지기 → 4효
3번째 던지기 → 3효
2번째 던지기 → 2효
1번째 던지기 → 1효 (초효)  ← 먼저
```

### 앱 구현 (todays-vibe)

```typescript
// src/data/iching.ts
export function throwCoins(): boolean {
  // 동전 3개: 합≥2 → 양효, 합<2 → 음효
  const sum = [0,1,2].reduce((acc) => acc + (Math.random() < 0.5 ? 1 : 0), 0);
  return sum >= 2;
}
```

> 현재 앱에서는 변효(變爻) 없이 단순 양/음 판별만 구현.  
> 변효 구현 시: 노양(3)·노음(0)은 기본괘에 반영 후 지괘(之卦)도 생성.

## 8. 이진법과 주역

라이프니츠(1703)가 선교사 부베로부터 전달받은 64괘 도해에서 이진법의 원리를 확인:

| 이진법 | 주역 |
|--------|------|
| 1 | 양효(—) |
| 0 | 음효(– –) |
| 6비트 이진수 | 대성괘(64가지) |
| 2^6 = 64 | 육십사괘 |

복희 선천괘도는 0(坤=000000)에서 63(乾=111111)까지의 이진 카운팅을 원형으로 배열한 구조다.
