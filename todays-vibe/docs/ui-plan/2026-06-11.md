### 1. Hero 섹션 — 오늘의 운세 카드

**현재**
```
당신만을 위한 오늘의 운세
사주·타로·꿈해몽 — ...

[카드 목록 바로 시작]
```

**개선 후**
```
✨ 오늘의 운세

  87점

오늘은 새로운 기회가 찾아오는 날입니다.
사람들과의 대화에서 좋은 에너지가 흐릅니다.

  연애운 ★★★★☆
  재물운 ★★★☆☆
  건강운 ★★★★★

  [자세히 보기]
```

**구현 방향**
- 점수(60~95 사이)와 한줄 멘트는 **날짜 기반 시드 난수**로 생성 (API 불필요, 서버 컴포넌트에서 처리)
- 3가지 운 별점도 동일하게 날짜 시드로 결정 → 하루 동안 같은 값 유지
- "자세히 보기" → 연애운 페이지 링크
- 파일: `src/components/home/TodayHeroCard.tsx` (신규)
- `page.tsx`에서 오늘 날짜를 시드로 넘겨줌

**점수 생성 예시**
```typescript
// 날짜 기반 시드 → 오늘 하루 동안 동일한 값
function getDailyScore(date: string): number {
  const seed = parseInt(date.replace(/-/g, ""), 10);
  return 60 + (seed % 36); // 60~95
}
```

### 2. Quick Menu — 인기 4종 상단 고정

**현재**: 카테고리 순서대로 나열, 인기/비인기 동일 취급

**개선 후**
```
┌──────┬──────┬──────┬──────┐
│  🃏  │  ☯  │  💕  │  💭  │
│ 타로 │ 사주 │ 연애운│ 꿈해몽│
└──────┴──────┴──────┴──────┘
```

**구현 방향**
- `fortunes.json`에 `featured: true` 필드 추가 (타로 원카드, 사주, 연애운, 꿈해몽)
- 홈 상단에 `QuickMenu` 컴포넌트 배치 (2×2 or 4열 그리드)
- 각 아이템: 아이콘 + 이름만 표시, 심플하게
- 파일: `src/components/home/QuickMenu.tsx` (신규)

### 3. 인기 운세 섹션

**현재**: 전체 카테고리가 동일 비중으로 나열

**개선 후**
```
🔥 많이 보는 운세

[타로 원카드]  [오늘의 별자리]  [연애운]
  → 2×1 큰 카드로 강조
```

**구현 방향**
- `fortunes.json`에 `popular: true` 필드 추가 (3~4개 선정)
- 전체 카드 목록 위에 별도 섹션으로 노출
- 카드 크기: 현재 카드보다 1.5배 크게 (col-span-1 → 더 큰 variant)
- 파일: `FortuneGrid.tsx` 상단에 `PopularSection` 추가

### 4. 카테고리 아코디언

**현재**: 5개 카테고리 전부 펼쳐짐 → 긴 스크롤

**개선 후**
```
📅 생년월일 운세  ▼  (기본 펼침)
   [카드들...]

🃏 타로/카드      ▼  (기본 펼침)
   [카드들...]

💕 궁합           ▶  (기본 접힘)
👤 관상/신체      ▶  (기본 접힘)
```

**구현 방향**
- 인기 카테고리 2개(생년월일, 카드)는 기본 펼침
- 나머지는 기본 접힘
- Framer Motion `AnimatePresence` + `height: "auto"` 애니메이션
- 상태는 `localStorage`에 저장 (새로고침해도 유지)
- `FortuneGrid.tsx` 내 카테고리 섹션에 토글 추가

### 5. 배경 시각 요소 강화

**현재**: SkyLayer가 이미 별 80개 + 구름 있음, 하지만 홈 카드 영역과 시각적 레이어 구분이 약함

**개선 방향**
- 카드 그리드 위에 아주 희미한 금빛 파티클 몇 개 (✦ ☆) 추가
  - opacity 4~6% 수준으로 거의 안 보이게
  - 위치 고정 (애니메이션 없어도 됨)
- 카드 hover 시 subtle한 glow 효과 강화
- Hero 카드에 별자리 패턴 SVG 배경 희미하게 추가

**구현 위치**: `SkyLayer.tsx` 또는 `page.tsx` 인라인 SVG

**현재**
```
[아이콘]
[제목]

[AI 뱃지]
```

**개선 후**
```
[아이콘]
[제목]
[설명]
[AI 뱃지]     [시작하기 →]
```

**구현 방향**
- `isReady` 카드에만 하단에 `시작하기 →` 텍스트 추가
- hover 시에만 나타나도록 (`group-hover:opacity-100`)
- PRO 뱃지: 현재 `PRO` → `✨ Premium` + 금색 그라데이션 테두리
